ImageReaderAI


With the implementation of AI, I wanted to create a more sophisticated way to analyze objects and allow users to maximize their information knowledge and time. Using my application will enable users to take a picture or choose one from the gallery to access an AI that will answer questions based on the user's prompt and image. Unlike other AI applications, its uniqueness lies in its instant image-based question-answering capability.

Why ImageReaderAI

There is artificial intelligence that allows you to ask questions based on an image or prompt, but there has yet to be an application that allows this to be done instantly. For example, let's say you are at a museum and like a painting. With ImageReaderAI, you can take a picture and get instant information on the details of the painting you choose.

  • Camera & App Set-Up

    There is a functional camera with options including camera rotation, flash On/Off, and allowing users to access pictures from their gallery.

    The user can re-take or save to library options.

    Library access when permissions accepted.

  • Implementing AI

    Based on the photo chosen by the user, the image will be redirected to Google Vision's AI. The user will be taken to a new screen with the chosen image and a text box so they can ask for information about the image.

    Since Bard is a new AI, I had to read a lot of documentation to learn how to implement this technology into my application.

  • Chat with AI (Bard)

    In this ChatBox, the user will be able to ask questions about the image that Bard, Google's AI, will answer.

    When the user prompts the question, Bard will reply, allowing them to go back and forth on the information the user wishes to know. This messaging screen will still contain the menu, allowing them to return to the camera to change the picture as the user can submit as, many pictures as they wish to input.

  • Firebase & Google Cloud

    I used Firebase and Google Cloud to create a secure authentication system for a React Native frontend. Firebase's authentication services simplified user sign-up, sign-in, and access control within the app. By integrating Google Cloud's infrastructure, I ensured a reliable and scalable authentication process, offering users a smooth and secure experience while interacting with the application.

    This setup not only provided a quick authentication solution but also established a robust foundation for potential future expansions within the Google Cloud environment.
